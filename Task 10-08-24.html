Q1: What is Difference between div and span?

A1:div: Block-level element; used for grouping larger sections and starts on a new line. span: Inline element; used for styling smaller parts within a line..

Q2: What are the semantic Elements?

A2: A semantic element clearly describes its meaning to both the browser and the developer..

Q3: What is difference between href attribute and src attribute?

A3:href: Links to a resource to be accessed on click. src: Embeds a resource directly into the page..

Q4: What is iframe element?

A4:An inline frame is used to embed another document within the current HTML document..

Q5: What is diffrence between xml and html?

A5: HTML: Used for creating and displaying web pages, focusing on the structure and presentation of content. XML: A markup language for storing and transporting data, focusing on data structure and custom tags without predefined semantics..
